# Efteling Onderhoudskalender

Deze tool laat je je abonneren op de Efteling Onderhoudskalender is je agenda.

### Endpoints

- `/feed`: Gebruik dit endpoint om de kalender te importeren in je kalenderapp

# Development

### Install dependencies

```
pnpm install
```

### Run locally

```
pnpm dev
```

### Deploy

```
pnpm deploy
```
